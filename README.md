# Executive Precision & Institutional AI Advisor 🏛️

Bu proje, bir "Resmi Kurum" (Institutional) ciddiyetinde ve görkeminde tasarlanmış, en üst düzey finansal danışmanlık portfolyosudur.

![Hero Preview](./hero-bg.png)

## 🌟 Özellikler

- **Executive Precision UI**: Ağırbaşlı, otoriter ve güven veren kurumsal tasarım dili.
- **6 Katmanlı Görsel Derinlik**: Sektörel otoriteyi temsil eden 6 adet yüksek çözünürlüklü profesyonel görsel entegrasyonu.
- **Institutional AI Advisor**: 225+ finansal mevzuat sorusuna yanıt verebilen akıllı asistan.
- **Bento-Box Infrastructure**: Modern ve organize bilgi hiyerarşisi.
- **Elite Performance**: Saf JavaScript ve CSS ile optimize edilmiş, ultra hızlı kullanıcı deneyimi.

## 🛠️ Teknolojiler

- **Frontend**: HTML5, CSS3 (Executive Design System), JavaScript (ES6+)
- **AI Engine**: Akıllı Anahtar Kelime & Mevzuat Eşleştirme Algoritması
- **Animations**: Intersection Observer API (Scroll-Reveal)
- **Design Concepts**: Glassmorphism, Bento Grid, Institutional Typography

## 📂 Dosya Yapısı

- `index.html`: Kurumsal iskelet ve içerik hiyerarşisi.
- `style.css`: Executive Precision tasarım sistemi ve responsive katmanlar.
- `main.js`: AI asistan mantığı, biletleme sistemi ve UI etkileşimleri.
- `data.js`: 225+ soruluk devasa finansal bilgi bankası.
